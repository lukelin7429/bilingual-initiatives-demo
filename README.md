# Bilingual Initiatives — Demo

A small, original demo site showing how a bilingual-English school program can be
presented on the web — **built and deployed by an AI agent** for a teacher workshop.

Five linked pages, one shared design system, audio pronunciation (Web Speech) and
scroll animations:

- **Home** — the four themes
- **English Club** — hands-on club activities
- **English Broadcast** — daily "phrase of the week" library
- **Speaking Corner** — Word of the Day + reward loop (interactive)
- **English Activities** — school-wide competitions, storytelling, song & dance

Hosted free on GitHub Pages. This is a demonstration template — content is original
and does not reproduce any individual teacher's site.
